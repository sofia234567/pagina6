LINGUAGO

LINGUAGO es una aplicacion movil innovadora que ofrece un metodo interativo y personalizado para aprender idiomas la aplicacion utiza tecnologia de inteligencia artificial para adaptar el proceso de aprendizaje a las necesidades y preferencias individuales de cada usuario Aprendizaje personalizado.
La aplicacion analiza el nivel de del usuario y crea un plan de aprendizaje adaptado a sus necesidades Contenido interactivo.LIGUAGO ofrece una variedadde jercic actividades para mantener al usuario motivado y comprometido.
